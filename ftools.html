FANTASTIC TOOLS
Here's a few tools and shaders I have created for Unity. Feel free to try them out, and please email me if you have any issues.
Kelvinator
Kelvinator lets you set real world light colour which follows the physical rules of Kelvin colour temperature.
Array Duplicate
ArrayDuplicate allows you to quickly make mutiple copies of an object at set intervals, in multiple dimensions. Similar to 3dsMax's Array, and Maya's Duplicate Special.
Foliage Shader
This shader is a Cut Out Standard shader that uses vertex animation to create a wind effect. It uses vertex colors to control the flexibility of the plant and has controls for wind strength, speed and turbulence.
Soft Body Wind Shader
Inspired by the fantastic art of Destiny 2, which always has some sci-fi nonsense blowing in the wind. Great for wires and cloth, this shader has controls for wind direction and speed. Uses vertex colour to control the stiffness across an object.
